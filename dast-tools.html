Dynamic Application Security Testing (DAST) Tools

Dynamic Application Security Testing (DAST) tools are essential for identifying security vulnerabilities in web applications while they are running. These tools simulate attacks on a web application to find vulnerabilities that could be exploited by attackers.

OWASP ZAP

OWASP ZAP is an open-source DAST tool that is designed to find security vulnerabilities in web applications during testing. It offers automated scanners as well as a set of tools for manual penetration testers.

Burp Suite

Burp Suite is a comprehensive solution for web application security checks. It includes an advanced set of tools for testing web security - all within a single product.

Invicti Security

Invicti Security (formerly Netsparker) is a web application security scanner that automates the process of identifying vulnerabilities and provides a proof of concept to confirm they are not false positives.
